## React/Preact - Boilerplate
My version of React/Preact Application Boilerplate with:

- Structured for Team Development
- Supports ES6
- State Container (redux)
- i18n (internationalization)
- Environment Variables 
- Server Side Rendering
- Progressive Web Apps
- Responsive 
- Testable 
- URL Router 
- Async Actions (rxJS)
- Dynamic CSS (styled-components)


## Quick Overview
```sh
git clone https://github.com/ciro-maciel/react-app-boilerplate.git

cd react-app-boilerplate/

npm install
npm run dev:client
```
